## VS Code Insiders Podcast Coverage

The podcast collection currently includes all 21 published episodes from the official feed.

- 9 episodes publish an official `transcript.txt` file and are stored with full transcript text.
- 12 episodes do not expose an official transcript file on the episode page and are stored as metadata-only placeholders.
- Episodes 18, 19, and 20 each include downloaded full transcript text.

## Notes

- Treat these files as source material, not polished prose.
- When a source does not publish an official transcript, keep the placeholder and avoid inventing one.
- Prefer the source-specific README for detailed indexes and per-file coverage.
