# Video Transcripts: @GitHub YouTube Channel

Source: https://www.youtube.com/@GitHub

This directory contains transcripts and metadata from GitHub's YouTube channel videos relevant to this guide. These serve as primary source material for verifying features, discovering new information, and linking to demos.

## How to Add Transcripts

Use the repository script to fetch a full timestamped transcript and append it to a stub file:

```powershell
python scripts/fetch-transcript.py <video_id> references/transcripts/github-channel/<stub>.md
```
