# GitHub Copilot in Visual Studio

[← Back to Foundations](../part-1-foundations.md)

---

[GitHub Copilot for Visual Studio](https://marketplace.visualstudio.com/items?itemName=GitHub.copilotvs) provides the full Copilot experience for .NET development.

## Capabilities

- **Agent mode** — autonomous multi-step task execution
- **MCP servers** — connect to external tools and data sources
- **Custom instructions** — `copilot-instructions.md` fully supported
- **Prompt files** — reusable task templates
- **Custom agents** — Preview
- **BYOK** — bring your own API keys (Visual Studio 18.0+)
- **Workspace indexing** — semantic search across the solution

## Primitive Support

| Primitive | Visual Studio Support |
|-----------|----------------------|
| Always-on Instructions | ✅ Supported |
| Custom agents | Preview |
| MCP servers | ✅ Supported |
| Prompt files | ✅ Supported |
| Agent mode | ✅ Supported |
| Agent skills | Not yet supported |
| Hooks | Not yet supported |

## Release Cadence

Releases are tied to the **Visual Studio release cycle** (approximately monthly). Copilot features ship as part of Visual Studio updates rather than through a separate extension.

## Getting Started

Visual Studio 2022 17.13+ or Visual Studio 2026 (18.0+) required. Copilot is included — sign in with your GitHub account.
