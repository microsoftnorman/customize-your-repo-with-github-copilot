# GitHub Copilot in Eclipse

[← Back to Foundations](../part-1-foundations.md)

---

[GitHub Copilot for Eclipse](https://marketplace.eclipse.org/content/github-copilot) provides code completion, Chat, agent mode, and MCP support. The plugin is [open source under MIT](https://devblogs.microsoft.com/java/ghc-eclipse-is-going-open-source/) and actively developed by Microsoft and the community.

## Primitive Support

| Primitive | Eclipse Support |
|-----------|----------------|
| Custom instructions | Preview |
| Custom agents | ✅ Supported |
| MCP servers | ✅ Supported |
| Agent mode | ✅ Supported |
| Agent skills | Not yet supported |
| Prompt files | Not yet supported |
| Hooks | Not yet supported |

## Eclipse Theia

[Eclipse Theia](https://theia-ide.org/) (the cloud/web IDE) also supports Copilot natively from version 1.68, including authentication, model selection, and agent features. Developers using Theia-based cloud environments get Copilot with their existing subscription.

## Release Cadence

Updates ship **approximately monthly**, with changelog entries posted to the [GitHub Blog](https://github.blog/changelog/label/copilot/). The open-source development model means community contributions can accelerate specific features.

## Getting Started

Requires plugin version 0.13.0+. Install from the [Eclipse Marketplace](https://marketplace.eclipse.org/content/github-copilot) and sign in with your GitHub account.
